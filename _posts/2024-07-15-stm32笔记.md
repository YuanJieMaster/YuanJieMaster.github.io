### STM32的寻址范围

32位单片机有32根地址线，单片机内存地址访问的存储单元是按字节编址的

STM32寻址大小2^32=4G（0x0000 0000 ~ 0xFFFF FFFF）



### 存储器映射

存储器只可以存储数据的设备，本身没有地址信息，对存储器分配地址的过程称为存储器映射

ST将4GB地址空间分成8个块



### 寄存器映射（重要）

单片机内部一种特殊的内存，可以实现对单片机各个功能的控制

STM32寄存器分类：内核寄存器、外设寄存器

#### PWM

PWM是用数字信号模拟模拟信号的过程，通过控制高低电位转换的周期，等效不同的电位，以下就是PWM控制发光二极管和电机的应用

输出模式根据是否复用和推挽开漏可以分为四种模式，开漏模式下上端开路，0输出低电平，1输出高阻抗

输入模式可以分为上拉、下拉、浮空、模拟模式，这几个模式之间的差别主要是在没有外界输入电平时将默认电平拉到Vdd或Vss

GPIO Pull-up/Pull-down ：选择上拉/下拉/浮空模式

```c
//读取IO的当前值
HAL_GPIO_ReadPin(GPIOx, GPIO_PIN);
```

既然要精确控制周期，那就需要了解计时器，时钟系统，时钟源和时钟树

STM32的时间系统在CubeMX中有一个对应的界面：时间配置界面





#### 串口

单片机通过串口和其他设备相连

stm32开发板可以通过 USB转TTL接口 和电脑相连

两个设备之间只需要两根线就可以进行数据交互，即各自的输出线（TX）连接对方的输入线（RX），最好再连第三条线--地线（GND），即共地

使用接口实现电脑和开发板之间的信息传输，到时候调试时将实时数据传到电脑上显示可以高效及时地发现问题

```c
//通过串口发送数据
HAL_StatusTypeDef HAL_UART_Transmit(UART_HandleTypeDef *huart, 
                                   uint8_t *pData,
                                   uint16_t Size,
                                   uint32_t Timeout);
发送的结果（成功/出错） HAL_UART_Transmit(串口句柄的指针, 
                                   要发送的数据的指针,
                                   要发送是数据的数量（以字节为单位）,
                                   超时时间（一般填HAL_MAX_DELAY)）; 
//通过串口发送数据是相似的
HAL_UART_Receive(UART_HandleTypeDef *huart, uint8_t *pData, uint16_t Size, uint32_t Timeout);
```



#### 呼吸灯

周期 =  arr + 1;

ccr = duty * (arr + 1); 一个周期内高电平的时间

```c
#include "math.h"
void main(void){
    HAL_TIM_PWM_Start(&htim1, TIM_CHANNEL_1); // 启动PWM的正常驱动
    HAL_TIMEx_PWMN_Start(&htim1, TIM_CHANNEL_1); // 启动PWM的互补驱动
    while(1){ //修改PWM的占空比
        float t = HAL_GetTick() * 0.001; // 获取当前时间
        float duty = 0.5 * sin(2*3.14*t) + 0.5; // 计算占空比
        uint16_t arr = __HAL_TIM_GET_AUTORELOAD(&htim1); // 获取ARR寄存器的值
        uint16_t ccr = duty * (arr+1); // 计算CCR寄存器的值
        __HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_1, ccr); // 将计算结果写入CCR
    }
}
```



#### PWM驱动舵机

首先需要知道舵机的特性

舵机的控制需要MCU产生一个20ms的脉冲信号，以0.5ms到2.5ms的高电平来控制舵机的角度

0.5ms------------0度：2.5% 对应函数中占空比为 250

1.0ms------------45度：5.0% 对应函数中占空比为 500

1.5ms------------90度：7.5% 对应函数中占空比为 750

2.0ms------------135度：10.0% 对应函数中占空比为 1000

2.5ms------------180度：12.5% 对应函数中占空比为 1250
